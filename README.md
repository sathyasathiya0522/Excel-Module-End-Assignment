# Ecommerce Sales Data Analysis – Project Summary

## Objective

The objective of this project was to clean, organize, analyze, and visualize ecommerce sales data using Microsoft Excel.

## Data Cleaning

The dataset was reviewed for missing values, duplicate records, formatting inconsistencies, and data quality issues. Missing values were handled using appropriate techniques such as averages, median values, and default values. Duplicate records were removed, and inconsistent Product IDs were standardized. Data validation was performed to ensure accuracy and consistency.

## Data Integration

Multiple tables were combined using VLOOKUP functions to create a unified dataset. Customer, product, and store information were merged into the sales data to support analysis and reporting.

## Statistical Analysis

Descriptive statistics including Mean, Median, Mode, Minimum, Maximum, Count, and Standard Deviation were calculated to understand data distribution and business performance.

## Pivot Table Analysis

Several Pivot Tables were created to summarize sales by category, region, payment type, customer, and month. These summaries helped identify trends and business insights.

## Dashboard and Visualization

Interactive charts and PivotCharts were developed to visualize sales performance. Slicers were used to filter data dynamically, creating an interactive dashboard for decision-making.

## Key Findings

* Sales performance varied across categories and regions.
* Payment methods showed different usage patterns.
* Monthly sales trends highlighted peak and low-performing periods.
* Pivot Table analysis helped identify top-performing products, customers, and stores.

## Conclusion

The dataset was successfully cleaned, integrated, analyzed, and visualized using Excel. The final dashboard and analytical reports provide meaningful business insights and support data-driven decision-making.
